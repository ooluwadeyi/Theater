# This Application takes the layout of the theater and the number of ticket request from the standard input. #
													    #
#############################################################################################################
Please take note of the following:

This application requires JRE version 1.8.0_131


1. double_click the runme.bat to launch appliction on console

2. For the layout, enter the number of seats in a row on the same line separate each section in  a row by a space.
3. For the ticket requests, enter the names of the Patrons and the number or request separated    by a space.
4. Write "end 0" at the end of the request to denote the end of the request
